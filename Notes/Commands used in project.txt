Initialize the npm 
    npm init
Install parcel as Dev dependencies that's why we need -D in command . npm contains all packages and repository through which we get dependencies .
    npm install -D parcel ()

Install all dependencies of projects 
    npm install

Run app using parcel by passing source
    npx parcel index.html

Install react
    npm install react

Install react-dom
    npm install react-dom

Build for production
    npx parcel build index.html

Run project using Scripts
    npm run start
    npm run build
